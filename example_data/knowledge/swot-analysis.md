# SWOT Analysis

A simple framing tool that organises internal capabilities and external conditions to inform strategy.

## The matrix

|              | Helpful        | Harmful     |
|--------------|----------------|-------------|
| Internal     | Strengths      | Weaknesses  |
| External     | Opportunities  | Threats     |

- **Strengths** — internal advantages: capabilities, IP, brand, distribution
- **Weaknesses** — internal limitations: cost structure, talent gaps, technical debt
- **Opportunities** — external trends to exploit: emerging segments, regulation, M&A targets
- **Threats** — external risks: new competitors, substitutes, macro shifts

## When to use

- Quick orientation in a new market or business
- Workshop tool for cross-functional alignment
- Pre-work for deeper frameworks (Five Forces, scenario planning)
- Annual strategy reviews

## Pitfalls

- Too generic if not pushed to specifics — "good people" is not a strength
- Confuses internal and external (e.g. listing competitor moves as a Weakness)
- Becomes a bullet list rather than analysis; pair each item with an implication
- No prioritisation: not all strengths matter equally
- Should drive a TOWS matrix (combining quadrants into actions) — otherwise it's diagnosis without prescription
