# Net Present Value (NPV)

NPV is the present value of a project's expected cash inflows minus the initial outlay. It measures absolute value creation.

## Formula

NPV = −C_0 + Σ (CF_t / (1 + r)^t)

where:
- C_0 = initial investment
- CF_t = net cash flow in period t
- r = discount rate (cost of capital or hurdle rate)

## Decision rule

- NPV > 0 → accept (creates shareholder value)
- NPV < 0 → reject
- NPV = 0 → indifferent

## When to use

- Comparing mutually exclusive projects
- Capital budgeting decisions
- Investment under known cash flows and discount rate

## Pitfalls

- Highly sensitive to the discount rate; document the choice
- Use incremental cash flows, not total revenue
- Ignore sunk costs; include only future, project-specific flows
- Don't forget tax effects, working capital, and terminal/salvage value
- For projects of different durations, use equivalent annual cost or extend horizons to match
